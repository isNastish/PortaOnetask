# PortaOnetasks
Скрипт на Python для  работы с файлом.

1) Скрипт для нахождения манимального числа в файле целых чисел,максимального,
	среденго арифметического,медианы.

2) А также для нахождения самой большой возрастающей последовательности,
	самой больлшой убывающей последовательности.

3)Если ваш исходный файл находиться не в той-же директории что и Python file, 
	в первой строке кода вам нужно указать путь к файлу.
